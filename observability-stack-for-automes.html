🗄️ VaultS3 — 对象存储底座
Rust 实现、S3 兼容、Apache 2.0、单二进制 ~15MB。提供 S3 API（:9000），AutoMES 应用文件存储 + GreptimeDB 数据存储共用一个实例。无需引入 MinIO（Java，~200MB+）或自建 Ceph。
📊 GreptimeDB — 统一可观测性后端
内置 Perses Dashboard（:4000/dashboard，含 Trace Gantt），原生 OTLP 接收（应用直发），SQL+PromQL 双查询，数据存 VaultS3（无状态容器）。
🔔 Alertmanager — 告警路由
Prometheus 生态标配，单二进制 ~20MB。从 GreptimeDB Prometheus API 拉取指标评估规则，Webhook 推送到飞书/钉钉/邮件。
